# 프로필 챗봇 (OpenAI 연동)

설정한 프로필(이름·성격·말투·전문분야)대로 대답하는 ChatGPT 스타일 채팅 앱입니다.

## 실행 방법
1. `index.html` 파일을 더블클릭해서 브라우저로 엽니다. (별도 설치/빌드 불필요)
2. 상단 **⚙️ 설정**에서 OpenAI API 키를 붙여넣습니다.
   - 키는 코드에 저장되지 않고 **현재 브라우저(localStorage)에만** 저장됩니다.
3. 성격 / 말투 / 전문분야를 고르고, 아래 입력창에서 대화하세요.

## 💰 돈이 함부로 나가지 않도록 한 장치
- **가장 저렴한 모델 `gpt-4o-mini` 고정** (비싼 모델로 못 바꿈)
- **답변 길이 상한**: `max_tokens = 300`
- **보내는 과거 대화 제한**: 최근 10개만 전송 (입력 비용 절감)
- **메시지 글자 수 제한**: 500자
- **키가 없으면 API 호출 자체를 안 함** → 돈이 들지 않음
- 화면에 **이번 세션 예상 비용**을 항상 표시
- 크레딧이 없으면 호출이 실패(429)하며, 이 경우 **비용은 청구되지 않습니다**

> 참고: OpenAI API는 쓴 만큼 과금되는 유료 서비스라 "완전 무료"는 없습니다.
> 다만 `gpt-4o-mini`는 매우 저렴해 보통 한 대화가 1원 미만입니다.

## ⚠️ 보안 주의
- API 키는 비밀번호와 같습니다. 공개 저장소/스크린샷에 노출하지 마세요.
- 채팅 등으로 한 번이라도 노출된 키는 [platform.openai.com](https://platform.openai.com/api-keys)에서
  **삭제(revoke) 후 새로 발급**받는 것이 안전합니다.
- 이 앱은 키를 파일에 저장하지 않으므로, `index.html`은 git에 올려도 키가 노출되지 않습니다.
